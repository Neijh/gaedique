Application de location de logement

Projet d'étude

Mon vieux grimoire est un site de référencement et de notation de livres ouvert par une petite chaîne de librairies.

Mon rôle

Développeur Back-end

  • Node.js
  • Express
  • MongoDB

Date

Mai 2024

L'objectif

Comprendre les défis

Implémenter une API sécurisée avec Node.js et Express avec MongoDB pour base de données.

Intégrer un système de notation et de classement.

Les solutions

Créer une expérience utilisateur moderne et réactive

Création du serveur : Mise en place d'un serveur avec Express, connecté à une base de données MongoDB.

Modelisation des données : Utilisation de Mongoose pour créer les modèles de données et implémenter les opérations CRUD (Créer, Lire, Mettre à jour, Supprimer) pour la gestion des livres et des notations.

Système d'authentification sécurisé : Développement d'un système d'authentification sécurisé pour les utilisateurs.

Gestion des images : Implémentation des fonctionnalités pour le téléchargement et l'optimisation des images.

Respect des bonnes pratiques du Green Code : Optimisation du code pour réduire l'empreinte écologique du site.

Ce que j'ai appris

Compétences développées

Maîtrise de Node.js et Express : Création et gestion de serveur web performants.

Utilisation de MongoDB et Mongoose : Modélisation des données et gestion des opérations CRUD.

Sécurité des données : Implémentation de systèmes d'authentification sécurisés.

Gestion des images : Téléchargement et optimisation des images pour améliorer l'expérience utilisateur.

Architecture MVC : Structuration efficace d'applications web.
