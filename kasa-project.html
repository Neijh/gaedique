Application de location de logement

Projet d'étude

Kasa est une entreprise de logements entre particuliers, qui permet aux utilisateurs de parcourir et de réserver des appartements

Mon rôle

Créer une expérience utilisateur moderne et réactive

  • React
  • Sass

Date

Avril 2024

L'objectif

Comprendre les défis

Le site de Kasa, initialement développé il y a plus de 10 ans en ASP.NET, présente un code legacy conséquent. Pour moderniser cete platforme, une refonte complète est nécessaire, passant à une stack entièrement en Javascript avec Node.js pour le back-end et React pour le front-end, Kasa en a également profité pour commander de nouvelles maquettes afin de rafraîchir l'apparence et l'expérience utilisateur du site.
Une refonte complète est nécessaire et l'expérience utilisateur ainsi que l'apparence du site on été repensées pour moderniser cette platforme

Les solutions

Créer une expérience utilisateur moderne et réactive

Gestion de la navigation : Pour configurer la navigation entre les différentes pages j'ai utiliser React Router, ce qui m'a permis de gérer efficacement le routage de l'application.

Manipulation des composants : Une fois les données extraites d'un fichier JSO, j'ai utilisé des states et des props afin de les propager à travers les composants et les afficher de manière cohérente sur les différentes pages.

Respect des maquettes : J'ai utilisé SASS pour adapter les maquettes fournies, styliser les composants et implémenter des animations CSS pour améliorer l'intéractivité et l'esthétique.

Ce que j'ai appris

Compétences développées

Maîtrise de React : J'ai acquis une compréhension approfondie de la création et de la gestion des composants React.

Gestion de la navigation avec React Router : J'ai appris à configurer et gérer la navigation entre les différentes pages d'une application web.

Utilisation de SASS : J'ai développé mes compétences en SASS pour gérer efficacement le CSS et styliser les composants de manière modulaire.

Implémentation d'animation CSS : J'ai appris à utiliser les animations CSS pour créer des interfaces utilisateur plus intéractives et engageantes.

Ce projet m'a permis de développer des compétences essentielles pour le développement web moderne. Telles que la maîtrise de React et React Router, ainsi que l'utilisation de SASS et l'implémentation d'animations CSS. Ces compétences sont cruciales pour créer des apllications web intéractives et visuellement attrayantes, très demandées dans l'industrie du développement web.
